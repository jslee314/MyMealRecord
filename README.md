# 서비스 소개 및 목적

- 아침 점심 저녁 먹은 음식을 기록하기위한 서비스

-  MVVM 아키텍쳐를 간단하게 정리하고 Firebase의 유용한 기능을 적용해보기 위함. 
-  특히 서버없이 Firebase로 데이터를 관리하기 위해

- ViewModel, LiveData(MVVM)
- cameraX
